#### MODULE NAME
       SL2_32

#### DEPENDENCIES
       SameBit

#### INPUTS
Name | Size(Bits)
-----|------------
A   |     32     

#### OUTPUTS
Name | Size(Bits)
-----|------------
Y   |     32     

#### DESCRIPTION
       Takes  a 32-bit input A, and shifts it left by 2.  Works for Two's Com-
       pliment values.
